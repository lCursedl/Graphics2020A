\doxysubsection{Detailed Description}
Class which contains a Singleton for a DirectX Device implementation. 

Instance can be initialized using a pointer for the class and setting it to N\+U\+LL. Then when needed, make a pointer of the class and assign its value given by \mbox{\hyperlink{class_c_device_a5e000c61a11d95a26d3f9b8ccf9ee1f0}{get\+Instance()}}.
