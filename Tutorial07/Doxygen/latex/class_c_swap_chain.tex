\doxysubsection{Detailed Description}
Class which contains a Singleton for a DirectX Swap Chain implementation. 

Instance can be initialized using a pointer for the class and setting it to N\+U\+LL. Then when needed, make a pointer of the class and assign its value given by \mbox{\hyperlink{class_c_swap_chain_a1f21ac2070e68387424ead065438373b}{get\+Instance()}}.
