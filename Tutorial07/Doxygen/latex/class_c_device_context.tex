\doxysubsection{Detailed Description}
Class which contains a Singleton for a DirectX Device Context implementation. 

Instance can be initialized using a pointer for the class and setting it to N\+U\+LL. Then when needed, make a pointer of the class and assign its value given by \mbox{\hyperlink{class_c_device_context_a2c86b335c65cff0b2e0a15154265b3dd}{get\+Instance()}}.
